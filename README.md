# Hangman Game 🎮

A simple hangman game: guess a word, don't die.

# Installation: 📝

1. open terminal:

```sh
cd project_directory
pod install
```
2. open hangman.xcworkspace file in directory
-----------
Mac M1 FIX: 

```sh
sudo gem install ffi
```

# Features: 🚀

• Different difficulty levels

• Custom design

• Gif animation

• Sound effects

• Hint button

• Mute music/soundFX button
